Install the following first:
1. `node` pkg version >= 9.5.0 
2. `npm` pkg version >= 5.6.0

Setup git. 

`git pull` from this repo. 

`cd deletemydata-web/`

`npm install`

Open two tabs in your terminal, 
1. `npm run dev`
2. `npm run start`

Go to `http://127.0.0.1:3000/` to see the website running
